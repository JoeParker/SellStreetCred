# Sell Street Cred

Mod for the game Cyberpunk 2077 which adds a new contact to the player's phone, allowing them to trade secrets in order to sell their Street Cred levels for money.

This mod requires the [Phone Extension](https://www.nexusmods.com/cyberpunk2077/mods/24949) framework.
